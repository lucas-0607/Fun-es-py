 Sistema Bancário Simples em Python

#📌 Descrição do Projeto

Este projeto consiste em um sistema bancário simples desenvolvido em Python. Ele permite realizar operações básicas como depósito, saque e transferência entre contas, incluindo validações para garantir a integridade dos dados.
O objetivo do projeto é demonstrar conceitos fundamentais de programação, como uso de funções, validação de entradas e organização de código.

---

## ▶️ Instruções de Execução

1. Certifique-se de ter o Python instalado (versão 3 ou superior)
2. Salve o código em um arquivo, por exemplo: `banco.py`
3. Abra o terminal ou prompt de comando
4. Navegue até a pasta onde o arquivo está salvo
5. Execute o comando:
       python banco.py


#  Principais Funcionalidades

*  *Depósito*

  * Adiciona um valor ao saldo
  * Valida se o valor é positivo

*  *Saque*

  * Remove um valor do saldo
  * Verifica se há saldo suficiente

*  *Transferência*

  * Transfere valores entre contas
  * Aplica uma taxa de 2% sobre o valor transferido
  * Valida saldo considerando a taxa

* 🛡️ *Validação de Dados*

  * Impede operações com valores inválidos
  * Utiliza tratamento de erros com `ValueError`
